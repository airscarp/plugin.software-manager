# Airscarp - Software Manager
This plugin allows you to manage programs installed on your server.

`This plugin is in BETA version.`

## Features
- [x] Software list 
- [x] Uninstall a software 
- [ ] List running programs
- [ ] Search in list
- [ ] Sort
- [ ] Install a software
